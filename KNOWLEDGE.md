## Token efficiency rules
- Tight prompts beat long prompts — 200 precise words > 2000 vague words
- Cache static context (company info, agent personas) — don't re-send every call
- Use Haiku for simple tasks, Sonnet for complex ones
- Load only what the current task needs — not the entire knowledge base
